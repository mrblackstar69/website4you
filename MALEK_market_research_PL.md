## 💡 Рекомендації для MALEK

### 1. Кольорова палітра (чорно-помаранчева)

```
Основні кольори:
├── Чорний:     #1A1A1A (текст, навігація)
├── Помаранчевий: #FF6B00 (CTA, акценти)
├── Темно-сірий: #2D2D2D (карта, footer)
└── Білий:      #FFFFFF (фон)

Градієнти:
├── Hero: linear-gradient(135deg, #1A1A1A 0%, #2D2D2D 100%)
└── CTA hover: linear-gradient(90deg, #FF6B00 0%, #FF8C00 100%)
```

**Чому це працює:**
- Помаранчевий вже використовує Riwal та частково Boels
- Чорний дає преміальний, індустріальний вигляд
- Контраст забезпечує читабельність та CTA виділення

**Первинний CTA (Помаранчевий):**
```css
.btn-primary {
  background: #FF6B00;
  color: #FFFFFF;

  border-radius: 4px;
  font-weight: 700;
  text-transform: uppercase;

  transition: all 0.3s ease;

.btn-primary:hover {
  background: #FF8C00;
  transform: translateY(-2px);
  box-shadow: 0 4px 12px rgba(255,107,0,0.4);
}
```

**Вторинний CTA (Обведений):**
```css
.btn-secondary {
  background: transparent;
  color: #1A1A1A;
  border: 2px solid #1A1A1A;

}
.btn-secondary:hover {
  background: #1A1A1A;
  color: #FFFFFF;
}
```

### 6. Типографіка

**Рекомендовані шрифти:**

```
Заголовки: Montserrat Bold/Black
├── H1: 48-64px, weight: 900
├── H2: 32-40px, weight: 700
└── H3: 24-28px, weight: 600

Текст: Inter або Open Sans
├── Body: 16-18px, weight: 400
├── Small: 14px, weight: 400
└── Caption: 12px, weight: 500
```

**Чому:**
- Montserrat - індустріальний, сучасний вигляд
- Inter - чудова читабельність на екранах
- Обидва безкоштовні (Google Fonts)

## 🎯 Висновок

Для преміум позиціонування MALEK на польському ринку рекомендую:

1. **Чорно-помаранчева палітра** виділить бренд серед конкурентів (жовтий Ramirent, синій Boels, червоний Kiloutou)

2. **Сильний Hero** з чіткою value proposition та візуальним фокусом на техніці

3. **Простота UX** - швидкий шлях до пошуку техніки та контакту

4. **Соціальний доказ** - статистика, відгуки, партнери

5. **Mobile-first** - значна частина B2B клієнтів шукає з телефону на будмайданчику
